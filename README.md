# 🏡 WanderNest

WanderNest is a full-stack web application inspired by Airbnb, built using Node.js, Express.js, MongoDB, and EJS. It enables users to create, explore, and manage property listings with a focus on clean architecture, modular design, and efficient server-side rendering. 🚧 *Currently under development.*

---

## 🔥 Key Highlights

* MVC-inspired project structure for maintainability
* RESTful routing and modular route organization
* Server-side rendering with EJS templates
* Scalable backend using Express and MongoDB (Mongoose)

---

## 🚀 Features

* 🏘️ Create, edit, and delete listings
* 📍 Location-based property data
* 🖼️ Image handling for listings
* 📂 Organized and modular codebase
* ⚡ Fast rendering using EJS

---

## 🛠️ Tech Stack

* **Frontend:** EJS, CSS, JavaScript
* **Backend:** Node.js, Express.js
* **Database:** MongoDB (Mongoose)
* **Version Control:** Git, GitHub

---

## 📂 Project Structure

```
WANDERNEST/
│── app.js              # Application entry point
│── package.json        # Project metadata & dependencies
│── schema.js           # Database schema definition

│── models/             # Mongoose models
│── routes/             # Route definitions
│── views/              # EJS templates
│── public/             # Static assets (CSS, JS)
│── utils/              # Utility/helper functions

│── init/               # Initialization scripts
│── node_modules/       # Installed dependencies (ignored)
```

## ⚙️ Installation & Setup

1. Clone the repository

```
git clone https://github.com/your-username/wandernest.git
```

2. Navigate to the project directory

```
cd wandernest
```

3. Install dependencies

```
npm install
```

4. Start the server

```
node app.js
```

5. Open in browser

```
http://localhost:8080
```
